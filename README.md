# Dotfiles

dotfiles for k1-c

## Environment
| | |
|---|-----|
|OS (Base)|Linux (Ubuntu ^20.04)|
|Shell| fish|

## Setup
```bash
# Clone this repository
git clone https://github.com/shunk-py/dotfiles.git
# All commands in Makefile
less Makefile
```
To setup all environments and configs at once:
```bash
make all
```
To install certain config:
```bash
# ex.
make neovide
```
